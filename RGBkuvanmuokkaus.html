Muokattu

Lataa kuva ja kokeile valmiita muunnoksia tai kirjoita omat funktiot.

Kirjoita funktiot muodossa esim. 255 - R tai G * 1.2.




Valmiita kokeiluja:

Ideoita omiin funktioihin:
  • 255 - R → kanavan kääntö eli negatiivi
  • R * 1.3 → kanavan kirkastus
  • R * 0.7 → kanavan himmennys
  • (R + G + B) / 3 → harmaasävyarvo
  • Math.pow(R / 255, 0.5) * 255 → korostaa tummia sävyjä
  • Math.sin(R / 255 * Math.PI) * 255 → sinimuotoinen muunnos
  • R > 128 ? 255 : 0 → kynnysarvo
Voit käyttää JavaScript-funktioita kuten Math.abs(), Math.pow(), Math.sqrt(), Math.sin(), Math.log(), Math.min() ja Math.max().
